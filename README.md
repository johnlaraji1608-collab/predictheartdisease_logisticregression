❤️ Heart Disease Prediction using Machine Learning
📌 Project Overview

Heart disease is one of the leading causes of death worldwide. Early prediction and diagnosis can help in reducing the risk and improving patient care.

This project aims to build a machine learning model that predicts whether a person is likely to have heart disease based on several medical attributes such as age, cholesterol level, blood pressure, and other health indicators.

The model is developed using Python in Google Colab and uses machine learning techniques to classify patients into heart disease or no heart disease categories.

🎯 Objectives

Analyze medical data related to heart disease

Perform data preprocessing and cleaning

Apply machine learning algorithms for prediction

Evaluate the performance of the model

Predict the presence of heart disease using patient data

📂 Dataset

The dataset contains several medical attributes used to predict heart disease.

Some important features include:

Age – Age of the patient

Sex – Gender of the patient

Chest Pain Type (cp) – Type of chest pain experienced

Resting Blood Pressure (trestbps) – Blood pressure while resting

Cholesterol (chol) – Serum cholesterol level

Fasting Blood Sugar (fbs) – Blood sugar level

Resting ECG (restecg) – Electrocardiographic results

Maximum Heart Rate Achieved (thalach)

Exercise Induced Angina (exang)

ST Depression (oldpeak)

Target – Presence of heart disease

Target Variable

0 → No Heart Disease

1 → Heart Disease
